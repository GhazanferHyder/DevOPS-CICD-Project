# DevOps CI/CD Project

## Project Overview

This project demonstrates an end-to-end DevOps CI/CD pipeline using a Python Flask application.

The objective of this project is to automate application build, containerization, deployment, infrastructure provisioning, and monitoring using modern DevOps tools.
